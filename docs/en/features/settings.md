# Settings

> Manage your account, team, billing, and API settings.

The Settings page allows you to configure your LiteStartup account, manage team members, update billing information, and access API credentials.

## Getting Started

Go to **Settings** from the sidebar to access all configuration options.

## Settings Tabs

| Tab | Description |
|-----|-------------|
| Usage | View current plan usage and limits |
| Billing | Manage subscription and payment |
| Team | Manage team members and roles |
| Send | Configure outbound email settings |
| Receive | Configure inbound email settings |
| Email Addresses | Manage email addresses |
| API Keys | Create and manage API keys |
| SMTP | Configure custom SMTP settings |

## Usage

View your current plan usage:

1. Go to **Settings** → **Usage**
2. See emails sent this month
3. View contact count
4. Check domain usage
5. Monitor limits

## Team

### Team Members

Manage your team:

1. Go to **Settings** → **Team**
2. View current team members
3. Invite new members
4. Update roles or remove members

### Roles

| Role | Permissions |
|------|-------------|
| Owner | Full access, billing, delete team |
| Admin | Manage team, domains, settings |
| Editor | Manage content, campaigns, growth features |
| Member | Send emails, manage contacts |
| Viewer | Read-only access |

### Invite Members

1. Click **Invite Member**
2. Enter email address
3. Select role
4. Click **Send Invite**

## Billing

### Current Plan

View your subscription:

- Plan name and price
- Billing cycle
- Usage this period
- Next billing date

### Upgrade/Downgrade

1. Go to **Settings** → **Billing**
2. Click **Change Plan**
3. Select new plan
4. Confirm change

### Payment Method

Update payment information:

1. Go to **Settings** → **Billing**
2. Click **Update Payment**
3. Enter new card details
4. Click **Save**

See [Pricing & Plans](pricing.md) for detailed plan comparison.

## API Keys

Manage API credentials:

1. Go to **Settings** → **API Keys**
2. View existing keys
3. Create new keys
4. Revoke unused keys

### Create API Key

1. Click **Create Key**
2. Enter key name (for reference)
3. Copy the key immediately
4. Store securely

> **Important**: API keys are shown only once. Store them securely.

## SMTP

Configure a custom SMTP server for sending emails:

1. Go to **Settings** → **SMTP**
2. Enter SMTP details:
   - **Name** - Display name for the SMTP config
   - **Host** - SMTP server hostname
   - **Port** - 587 (TLS) or 465 (SSL)
   - **Username** - SMTP username
   - **Password** - SMTP password
3. Click **Save**

> **Note**: Encryption is automatically set to TLS for port 587 and SSL for port 465.

## Best Practices

- **Strong passwords** - Use unique, complex passwords
- **Review team access** - Audit permissions regularly
- **Rotate API keys** - Change keys periodically
- **Monitor usage** - Check billing and usage regularly
