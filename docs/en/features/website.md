# Website

> Create landing pages and waitlist pages with AI Website Builder.

The AI Website Builder allows you to create professional landing pages, waitlist pages, and more without coding. Simply describe what you want, and AI generates the page for you.

## Getting Started

1. Go to **Website** from the dashboard
2. Click **Create Page** to start a new page
3. Describe your page to AI or choose a template
4. Customize and publish

## Page Types

| Type | Description | Use Case |
|------|-------------|----------|
| Landing Page | Marketing page for your product | Product launches, campaigns |
| Waitlist Page | Collect signups before launch | Pre-launch, beta access |
| About Page | Company or team information | Company profile |
| Contact Page | Contact form and information | Customer inquiries |

## Creating a Page

### With AI

1. Click **Create with AI**
2. Describe your page (e.g., "A landing page for my SaaS product that helps startups send emails")
3. AI generates the page design and content
4. Review and customize
5. Click **Publish**

### From Template

1. Click **Choose Template**
2. Browse available templates
3. Select a template
4. Customize content
5. Click **Publish**

## Customization

### Edit Content

- Click on any text to edit
- Update headlines, descriptions, and CTAs
- Add or remove sections

### Styling

- Change colors to match your brand
- Update fonts
- Adjust spacing and layout

### Images

- Upload your own images
- Use AI-generated images
- Choose from stock photos

## Custom Domain

Host your page on your own domain:

1. Go to **Domains** and add your domain
2. Add CNAME record pointing to `yourdomainid.litestartup.net`
3. Select domain in page settings
4. Your page is live at `www.yourdomain.com`

## Waitlist Features

For waitlist pages:

- **Email collection** - Capture visitor emails
- **Referral tracking** - Track who referred whom
- **Position display** - Show waitlist position
- **Auto-notifications** - Notify when invited

## Analytics

Track page performance:

- Page views
- Unique visitors
- Conversion rate
- Traffic sources

## Best Practices

- **Clear headline** - Communicate value immediately
- **Strong CTA** - Make the action obvious
- **Mobile-friendly** - Pages are responsive by default
- **Fast loading** - Optimized for performance
- **A/B test** - Try different versions
