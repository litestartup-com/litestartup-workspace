# Blog

> Create and manage a blog for your website with AI assistance.

The Blog feature allows you to create and publish blog posts to engage your audience, improve SEO, and establish thought leadership.

## Getting Started

1. Go to **Blog** from the dashboard
2. Click **Create Post** to write a new article
3. Use AI to help generate content
4. Publish and share

## Creating a Blog Post

### Write with AI

1. Click **Create Post**
2. Enter a topic or title
3. Click **Generate with AI**
4. AI creates a draft based on your topic
5. Edit and refine the content
6. Click **Publish**

### Write Manually

1. Click **Create Post**
2. Enter title and content
3. Add images and formatting
4. Set category and tags
5. Click **Publish**

## Post Editor

### Formatting Options

- **Headings** - H1, H2, H3 for structure
- **Bold/Italic** - Emphasis
- **Lists** - Bullet and numbered lists
- **Links** - Add hyperlinks
- **Images** - Upload or embed images
- **Code blocks** - For technical content

### Markdown Support

Write in Markdown for faster editing:

```markdown
# Heading 1
## Heading 2

**Bold text** and *italic text*

- Bullet point
- Another point

[Link text](https://example.com)
```

## Managing Posts

### Post Status

| Status | Description |
|--------|-------------|
| Draft | Work in progress, not visible |
| Published | Live and visible to readers |
| Scheduled | Will publish at set time |

### Edit Post

1. Click on a post to open
2. Make changes
3. Click **Update**

### Delete Post

1. Click **Delete** on a post
2. Confirm deletion

## Categories and Tags

Organize posts for easy navigation:

- **Categories** - Broad topics (e.g., "Product Updates", "Tutorials")
- **Tags** - Specific keywords (e.g., "email", "marketing", "tips")

## Custom Domain

Host your blog on your domain:

1. Configure domain CNAME record
2. Set blog URL in settings
3. Blog is accessible at `blog.yourdomain.com`

## SEO Features

- **Meta title** - Custom title for search results
- **Meta description** - Summary for search results
- **URL slug** - Clean, readable URLs
- **Open Graph** - Social media preview

## Best Practices

- **Consistent schedule** - Publish regularly
- **Quality content** - Focus on value for readers
- **Use images** - Break up text with visuals
- **Optimize for SEO** - Use keywords naturally
- **Promote posts** - Share via email and social media
