Frequently Asked Questions

Quick answers to common questions.

What is LiteStartup?

LiteStartup is an all-in-one email infrastructure platform for startups — covering transactional email, marketing campaigns, workmail, and more.

?

Is there a free plan?

Yes! Our free plan includes essential features to get started. Upgrade to Pro for unlimited sending, advanced analytics, and priority support.

?

How do I set up a custom domain?

Add your domain in the dashboard, then configure DNS records (MX, SPF, DKIM). We support automatic setup via Cloudflare API or manual configuration.

?

Can I migrate from another provider?

Absolutely. We support importing contacts and templates. Check our documentation for migration guides.

?

Do you offer an API?

Yes, we provide a full REST API for sending emails, managing contacts, and more. See our API reference.

?

How can I report a bug?

Use the contact form above and select "Bug Report" as the subject. Include steps to reproduce the issue and we'll prioritize the fix.
